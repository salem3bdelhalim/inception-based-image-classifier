# 👟 Shoe vs Sandal vs Boot Classification using CNN

This project uses Deep Learning models (CNN + Inception) to classify footwear images into:

* Shoe
* Sandal
* Boot

---

## 📁 Dataset

Dataset used:
https://www.kaggle.com/datasets/hasibalmuzdadid/shoe-vs-sandal-vs-boot-dataset-15k-images

---

## 🧠 Model

* Convolutional Neural Network (CNN)
* Inception Architecture
* Factorized Convolutions 
* Image Augmentation

---

## 📊 Results

| Metric    | Score |
| --------- | ----- |
| Accuracy  | 0.9820  |
| Precision | 0.9820   |
| Recall    | 0.9820  |
| F1 Score  | 0.9820
   |
